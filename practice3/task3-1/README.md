\# Задание 3-1 (Вариант 5): Двусторонний обмен через два канала



\## Описание

Программа создаёт два неименованных канала (`pipe`) для двустороннего обмена между родительским и дочерним процессами. Родитель отправляет целое число, дочерний вычисляет его квадрат и возвращает результат.
